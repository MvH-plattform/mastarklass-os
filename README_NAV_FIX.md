# Mästarklass OS 9.8.0 – navigeringsfix

Denna korrigering löser felet där startsidan låg kvar ovanpå alla undermenyer.

## Uppladdning
1. Ladda upp `live_loader_9_8_0.js` till repositoryts rot.
2. Bekräfta att GitHub ersätter filen med samma namn.
3. Gör Commit.
4. Vänta tills GitHub Pages är färdig.
5. Stäng appfliken helt och öppna appen igen.

`index.html` behöver inte ändras.
