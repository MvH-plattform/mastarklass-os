# Mästarklass OS 3.1 – Portfolio Intelligence

## Nytt
- moderna innehavskort för samtliga verifierade poster
- antal eller andelar
- GAV
- marknadsvärde
- anskaffningsvärde
- resultat i kronor och procent
- konto, tillgångsslag, sektor, land och valuta
- lokal Mästarklass Score 0–100
- status: Stark, Stabil, Neutral, Bevaka eller Datagap
- sökning, filter och sortering

## Databas

- innehav med lokal score: 101

## Viktigt
Mästarklass Score i 3.1 är en lokal regelbaserad modell. Den bygger på datatäckning, sektor, utveckling mot kostnadsbas och enkel koncentration. Den är inte ett externt kvalitetsbetyg, livevärdering eller personlig investeringsrådgivning.
