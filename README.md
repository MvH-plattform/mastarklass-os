# Mästarklass OS 8.9 – Portfolio Intelligence & Safety Layer

## Nytt
- Sandbox/testläge för fiktiva köp och försäljningar
- inga sandboxförändringar skrivs till Private Vault
- Portfolio Impact för vikt, sektor, valuta, portföljvärde och Portfolio IQ
- överföring av ett godkänt scenario till det verkliga transaktionsformuläret
- Backup Intelligence med ålder, antal ändringar och rekommenderad åtgärd
- smart transaktionshistorik med sökning och filter
- Data Integrity Engine mellan Private Vault, Unified Data Provider och modulerna
- lokal portföljtidslinje
- AI Decision Coach 1.0
- fortsatt lokal och privat datalagring
- externa marknadsdata, bankkoppling och handel fortsatt avstängda

## Testa ett fiktivt köp
1. Öppna **Mer → Portfolio Intelligence & Safety 8.9**.
2. Välj Sandbox.
3. Välj innehav, antal, pris, valuta, valutakurs och courtage.
4. Läs påverkan på Portfolio IQ, vikt, sektor och valuta.
5. Rensa testet eller välj **Överför till verklig transaktion**.
6. Ingenting ändras i Private Vault förrän du sparar i det verkliga transaktionsformuläret.
