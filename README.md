## Databas

## Viktigt
Analysen använder endast lokalt lagrad och verifierad data. Poster utan marknadsvärde räknas inte in i viktberäkningarna. Fond- och ETF-överlapp på underliggande bolagsnivå byggs i ett senare steg.
