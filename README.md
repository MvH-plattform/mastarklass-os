# Mästarklass OS 9.0.1 – Investment Credit Center

## Omfattning
Denna version följer endast kredit som är direkt kopplad till värdepappersportföljen, exempelvis Avanza Kredit. Lön, hushållsekonomi, bolån, billån och andra privata skulder ingår inte.

## Nytt
- kreditlimit
- utnyttjad värdepapperskredit
- tillgängligt kreditutrymme
- registrerad ränta
- beräknad dags-, månads- och årsränta
- utnyttjandegrad
- belåningsgrad mot registrerat belåningsvärde
- säkerhetsnivå
- amorterings- och kredithistorik
- simulering av köp med kredit
- simulering av försäljning där likviden amorterar krediten
- lokal lagring i Private Vault
- automatisk lokal backup efter kreditförändringar
- förberedelse för Portfolio IQ 2.0 och AI Decision Coach

## Exempel: Alibaba säljs och minskar krediten
1. Registrera försäljningen i Transaktioner.
2. Öppna **Investment Credit Center → Förändring**.
3. Välj **Amortera kredit**.
4. Välj kopplingen **Försäljningslikvid**.
5. Ange nettobeloppet som faktiskt minskade krediten.
6. Spara.

Kreditsaldot, utnyttjandegraden och den beräknade räntekostnaden uppdateras automatiskt.
