# Mästarklass OS 9.8.2 – Portfolio Live Command

Ett sammanhängande produktsteg med två prioriteringar: snabbare användning av den långa Portföljvyn och fortsatt säker liveaktivering.

## Nytt
- Sticky snabbnavigering i Portfölj: Översikt, Konton, Tillgångsslag, Största innehav, Datakvalitet, Innehavsanalys, Beslut och Rating.
- Aktiv sektion markeras automatiskt under scrollning.
- Flytande knapp tillbaka till toppen.
- Live Command på Marknad med status för ECB-valutor, mappningar, sparade kurser och connector.
- Direktknapp till Live Portfolio samt säker omhämtning av lokal live-status.
- Read-only-principen bevaras: live-data ändrar aldrig antal, GAV, transaktioner eller Private Vault.

## GitHub-uppladdning
Ladda upp samtliga filer i paketet till repositoryts rot och ersätt filer med samma namn. Ingen manuell kodredigering behövs.

Efter commit: invänta GitHub Pages, stäng appfliken helt och öppna den på nytt.
