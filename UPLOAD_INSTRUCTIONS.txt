MÄSTARKLASS OS 10.0 — UPPLADDNING

1. Öppna GitHub-repositoryt mastarklass-os.
2. Välj Add file → Upload files.
3. Ladda upp ALLA filer i detta paket samtidigt.
4. Välj Commit changes.
5. Vänta 1–3 minuter på GitHub Pages.
6. Stäng gamla appflikar helt och öppna sidan på nytt.

VIKTIGT:
- Du ska inte redigera index.html manuellt.
- Gamla filer kan ligga kvar utan att köras eftersom 10.0 inte refererar till dem.
- Den nya sw.js rensar äldre cache automatiskt.
- När 10.0 är verifierad kan gamla README_*, patch-, recovery-, live_loader_* och version_guard_* rensas i ett separat städsteg.
